# VoiceChat Demo Prototype

## What It Is
A mobile-first React prototype of a Telegram/WhatsApp-style chat app focused on opt-in voice conversations and training-data approval.

## Goal
Demo the end-to-end UX flow (home → chat → call → post-call summary approval → earnings) without requiring a real backend.

## Stack
- **Framework:** Vite + React
- **Rendering:** Standard React UI (no WebGL)
- **Animation:** CSS interactions/transitions
- **Libraries:** lucide-react

## Inspiration / References
- Sam-provided UI references for Home, Chat, Call, Post-call Summary, Earnings
- Design system: light/minimal/spacious, Satoshi typography, slate/blue palette

## Performance Targets
- Target FPS: 60
- Target devices: mobile-first (iPhone-sized viewport)
- Max bundle size: not optimized yet (demo-only)
